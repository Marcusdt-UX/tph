How to set up USB / Wireless debugging
  1. Go to Settings → About phone and tap Build number 7 times to unlock Developer Options.
  2. Go to Settings → Developer options and enable USB debugging.
  3. For wireless: enable Wireless debugging and note the IP address.
  4. Connect your cable (or enter the IP above), then accept the authorization prompt on your phone.
  5. For DeX: enable DeX mode in Settings → Samsung DeX, or tap the DeX button in the taskbar below.
